# Literature Review

This folder stores structured GNN paper notes, visual explanations, and method-oriented reading summaries.

Each note should explain:

- the problem the paper addresses
- the model architecture
- the key mathematical ideas
- the training and evaluation logic
- the main contribution and limitations
- how the idea may transfer to biomedical GNN research

## Folder Rules

Keep this folder focused on literature and methods.

Do not store:

- datasets
- model checkpoints
- generated result tables
- large experiment figures
- project-specific intermediate outputs
