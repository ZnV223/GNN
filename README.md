# GNN Learning Notes

This repository stores learning notes, paper-reading summaries, and visual explanations for graph neural networks (GNNs).

The goal is to keep conceptual study materials separate from project-specific datasets, experiment outputs, model checkpoints, and result tables.

## Scope

This repository is for:

- GNN learning notes
- paper summaries
- method explanations
- diagrams and conceptual visualizations
- reusable methodological ideas

This repository is not for:

- raw datasets
- trained model checkpoints
- generated experiment outputs
- large figures or result tables
- private or sensitive project files

## Reading Path

Suggested learning order:

1. Understand graph theory basics and message passing.
2. Study core models such as GCN, GraphSAGE, GAT, and GIN.
3. Read application-oriented papers such as MeshGraphNets.
4. Compare model assumptions, graph construction strategies, and evaluation protocols.
5. Translate useful ideas into biomedical GNN research designs.
